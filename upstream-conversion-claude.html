Upstream Conversion
mammoth & Pandoc

Converts .docx intelligence reports into structured markdown optimised for Claude's context window. This is an upstream pre-processing step — convert here, upload the .zip to Claude, and it spends its full token budget on analysis instead of parsing binary document formats at runtime.

How it works
Embedded images are extracted as separate files and referenced by path in the markdown. The output is a single .zip file.

Upload the .zip directly to Claude — do not unzip first. Claude will unpack it automatically, read the markdown files natively in its context window, and interpret extracted images as visual inputs. No base64 bloat, no wasted tokens on runtime conversion.
